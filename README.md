## Estructura del Proyecto

Este repositorio contiene la solución de la Fase 1 del curso "Programación con LLMs" de la Universidad de Antioquia.

### Contenido

- **myquestions/**: Directorio con las 4 preguntas (retos) y sus respectivos generadores de casos de uso aleatorios.
  - `question-XXXX.txt`: Enunciado de cada pregunta
  - `question-XXXX-usecase-generator.py`: Generador de casos de uso aleatorios para validación

### Retos

1. **Reto 1**: Preprocesamiento - Normalización Adaptativa por Ventana Temporal
2. **Reto 2**: Regresión - Predicción de Tiempo de Ciclo con Error Asimétrico
3. **Reto 3**: Clasificación - Detección de Adulteración en Aceite de Oliva
4. **Reto 4**: Clustering + Reducción de Dimensionalidad - Segmentación de Zonas Sísmicas
